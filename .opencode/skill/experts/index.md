## Usage Examples

### Single Expert
```
I need the prisma-expert to help design a schema for my UPSC app with users, notes, and quizzes.
```

### Multiple Experts
```
Use the nextjs-expert and tailwind-expert together to create a responsive dashboard component.
```

### Security Review
```
Ask the owasp-top10-expert to review this authentication implementation for vulnerabilities.
```

### Architecture Decision
```
I need the docker-expert and terraform-expert to help containerize and deploy this Next.js app to AWS.
```
