## 📊 DATABASE SCHEMA (Key Tables)

### Core Tables
```sql
-- Users & Authentication
users (id, email, name, role, subscription_tier, preferences, created_at)
user_subscriptions (id, user_id, plan_id, status, current_period_end)

-- Content
notes (id, user_id, topic, content, ai_generated, video_url, created_at)
quizzes (id, topic, difficulty, questions_json, ai_generated)
quiz_attempts (id, user_id, quiz_id, score, time_taken, answers_json)
current_affairs (id, title, content, source, date, relevance_score, video_url)
lecture_videos (id, topic, duration, video_url, transcript, manim_scenes, status)

-- Learning Progress
study_plans (id, user_id, daily_tasks, completed, streak_count)
study_sessions (id, user_id, topic, duration, performance_score)
user_bookmarks (id, user_id, content_type, content_id, notes, revision_date)
revision_schedule (id, user_id, topic, next_revision, confidence_score)

-- Analytics
user_analytics (id, user_id, topic, time_spent, quiz_scores, confidence_meter)
syllabus_progress (id, user_id, topic_id, completion_percentage, performance)

-- Video Generation
video_jobs (id, user_id, topic, job_type, status, output_url, created_at)
video_templates (id, name, type, remotion_config, manim_scenes)

-- Monetization
payments (id, user_id, amount, plan, status, razorpay_id)
usage_credits (id, user_id, credit_type, balance, expires_at)
```

## 🔐 SECURITY REQUIREMENTS

### Authentication & Authorization
- [ ] NextAuth with Supabase adapter
- [ ] JWT tokens with refresh rotation
- [ ] Role-based access control (user, admin, super_admin)
- [ ] Row-level security (RLS) on all tables
- [ ] API rate limiting per user/subscription tier

### Data Protection
- [ ] Encryption at rest (Supabase native)
- [ ] Encryption in transit (TLS 1.3)
- [ ] Secure environment variables (never commit)
- [ ] Audit logging for all sensitive operations
- [ ] GDPR compliance (data export, deletion)

### Video Security
- [ ] Signed URLs for video access
- [ ] DRM for premium content (Phase 3)
- [ ] Watermarking for downloaded content
- [ ] Concurrent stream limits
