## 1. Feature Overview

### 1.1 Purpose
Motivate students through:
- **XP System**: Earn points for tasks, accuracy, and streaks. Redeem in XP Shop.
- **Achievements**: Unlockable badges for milestones (e.g., "Speed Demon" for fast tests).
- **Leaderboards**: Weekly, Monthly, and All-time rankings.
- **Streaks**: Visual tracking of daily consistency with protection.

### 1.2 User Stories

#### US-F13-01: Earn & Spend XP
**As a** student  
**I want to** earn XP and buy rewards  
**So that** I stay motivated  

**Acceptance Criteria:**
- XP for: Task (+10), MCQ (+5), Streak (+20), Perfect Score (+50)
- Shop: Profile Frames, Themes, Extra Features
- Balance updates in real-time

#### US-F13-02: Unlock Badges
**As a** student  
**I want to** unlock achievements  
**So that** I feel proud  

**Acceptance Criteria:**
- Badges: First Blood, 7-Day Streak, Top 10, Speedrun
- Profile display with tooltips
- Notification on unlock

#### US-F13-03: Compete on Leaderboard
**As a** student  
**I want to** see my rank  
**So that** I can compete  

**Acceptance Criteria:**
- 3 Tabs: Weekly, Monthly, All-time
- Highlight own rank
- Filter by Friends

#### US-F13-04: Streak Freezes
**As a** student  
**I want to** protect my streak  
**So that** I can take a break  

**Acceptance Criteria:**
- 100 XP = Streak Freeze (1 day)
- Max 2 freezes stacked
